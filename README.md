# [E-Commerce 시스템]

## 프로젝트 소개

이 프로젝트는 서버구축을 위한 설계 작업을 위한 프로젝트 입니다.

## 주요 기능

* 사용자 및 잔액 관리
* 상품 관리
* 쿠폰 관리
* 주문 및 결제
